# Запуск проекта

## Инструкция по запуску

1. Установите зависимости (если ещё не установлены):
   cd legal-website
   npm install

2. Запустите сервер разработки:
   npm run dev

3. Откройте браузер по адресу:
   http://localhost:3000

## Остановка сервера

Для остановки сервера нажмите Ctrl+C в терминале

## Сборка для продакшена

npm run build

## Предпросмотр продакшн-сборки

npm run preview
