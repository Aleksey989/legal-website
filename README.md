# Юридический сайт с онлайн оплатой

Одностраничный сайт (Landing Page) для юридической компании «Правовая Защита» с интеграцией онлайн-оплаты.

## 🚀 Запуск проекта

### Требования
- Node.js (версия 14 или выше)
- npm

### Установка зависимостей
`ash
cd legal-website
npm install
`

### Запуск сервера разработки
`ash
npm run dev
`

Сайт будет доступен по адресу: http://localhost:3000

### Сборка для продакшена
`ash
npm run build
`

### Предпросмотр продакшн-сборки
`ash
npm run preview
`

## 📋 Функционал сайта

### Основные секции:
1. **Навигация** — фиксированная шапка с якорными ссылками
2. **Герой-секция** — главный заголовок с призывом к действию и статистикой
3. **Услуги** — 6 карточек с юридическими услугами и ценами
4. **О компании** — информация о фирме и преимуществах
5. **Команда** — карточки 4 юристов
6. **Отзывы** — 6 отзывов клиентов с рейтингом
7. **Форма записи** — форма на консультацию с выбором услуги
8. **Оплата онлайн** — интеграция с PayPal, банковскими картами и ЮMoney
9. **Контакты** — адрес, телефон, email, карта, соцсети
10. **Футер** — ссылки, соцсети, копирайт

### Технологии:
- **React 18** — библиотека для создания интерфейсов
- **Vite** — сборщик и сервер разработки
- **Bootstrap 5** — CSS-фреймворк для стилизации
- **Bootstrap Icons** — иконки

### Дизайн:
- Цветовая схема: тёмно-синий (#1a237e), белый, золотой акцент (#d4af37)
- Шрифты: Playfair Display (заголовки), Roboto (текст)
- Адаптивный дизайн для всех устройств
- Плавные анимации при скролле и наведении

### Платёжная система:
- **PayPal** — готовая интеграция (требуется настройка API-ключей)
- **Банковские карты** — заглушка для интеграции (Stripe, CloudPayments)
- **ЮMoney** — заглушка для интеграции

## 🔧 Настройка оплаты

### PayPal
Для реальной работы PayPal необходимо:
1. Зарегистрироваться на [PayPal Developer](https://developer.paypal.com/)
2. Создать приложение и получить Client ID
3. Добавить скрипт PayPal SDK в index.html

### Stripe (для банковских карт)
1. Зарегистрироваться на [Stripe](https://stripe.com/)
2. Получить publishable key
3. Установить библиотеку: 
pm install @stripe/stripe-js
4. Интегрировать в компонент Payment

### ЮMoney
1. Зарегистрироваться на [ЮMoney](https://yoomoney.ru/)
2. Получить номер кошелька или создать форму приёма платежей
3. Интегрировать форму в компонент Payment
